GUITAR PRO IMPORT GUIDE

Even though you can manually author all pro guitar content in EOF, many people prefer to import a Guitar Pro file. Some even author an arrangement from scratch in Guitar Pro and then import it. How you decide to use Guitar Pro files, if at all, is completely at your discretion. EOF is able to import Guitar Pro 5 (.gp5) and older versions of the format. The newer Guitar Pro formats (.gpx or .gp) aren’t understood by EOF at this time, but I hope to add support for .gp in the future.

Guitar Pro files have been a popular tablature format for many years, so they can be found on the Internet easily. One popular website is https://www.ultimate-guitar.com/

As discussed earlier in this tutorial, if you have exported Guitar Pro XML timing from Go PlayAlong, you can import that and don’t have to manually beat sync the project. You can do this by using “File>Import>Guitar Pro” and browsing for the XML file you exported from Go PlayAlong. The XML file internally references the Guitar Pro file by name, so as long as the Guitar Pro file is named correctly, in the same folder as the Go PlayAlong XML file and in a supported format (.gp5 or older), EOF should be able to import the timing and then import the tablature.

Otherwise it’s best if you beat-sync the project (as described in the [CREATING AND BEAT SYNCING A PROJECT] section) before importing a Guitar Pro file. If your project is beat-synced, change to the desired pro guitar track (using Song>Track or the CTRL+Tab shortcuts) and use “File>Import>Guitar Pro”.

EOF’s Guitar Pro import is reasonably reliable, but there are occasionally bugs such as when there are complex measure repeat patterns in the tablature or when a Guitar Pro 5 file was exported from a newer version of Guitar Pro (sometimes the GP5 file is written incorrectly due to a bug with Guitar Pro 6, 7 or 8). Usually though, if the file can open in the official Guitar Pro 5, or in TuxGuitar (a free guitar tablature editing program, https://www.tuxguitar.app/), and it appears accurate, EOF should be able to import it accurately as well. As with other imports, test the beat sync during playback with the metronome (Edit>Metronome, or press the M key). Test note sync during playback with clap ("Edit>Claps" or press the C key). If you’re satisfied with the content, proceed to export to Immerrock format [IMMERROCK EXPORT GUIDE]
